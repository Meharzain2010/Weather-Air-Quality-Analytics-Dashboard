# E-Commerce Sales Analytics Dashboard

## Overview

This project demonstrates **end-to-end data analysis and business intelligence dashboarding** using Power BI.

The goal is to analyze e-commerce sales data and extract insights related to **revenue, profit, customer behavior, and product performance**.

The dashboard enables stakeholders to make **data-driven decisions** through interactive visualizations.

---

## Tools Used

* Power BI
* Python (Pandas, NumPy)
* SQL
* Data Cleaning & Transformation

---

## Features

* Interactive dashboard with **dynamic filters (Quarter, Category)**.
* Analysis of **revenue (438K), profit (37K), and quantity sold**.
* Identification of **top-performing states and customers**.
* Category-wise analysis (Clothing, Electronics, Furniture).
* Payment mode distribution (COD, UPI, Credit Card, etc.).
* Monthly profit trends and seasonal insights.
* Sub-category level performance analysis.

---

## Dataset

The dataset consists of e-commerce sales records including:

* Order Amount
* Profit
* Quantity
* Customer Details
* Product Categories
* Payment Modes
* Geographic Information

---

## Analysis Performed

* **Monthly Profit Trend Analysis**
* **Top States by Sales**
* **Top Customers by Revenue**
* **Category & Sub-category Performance**
* **Payment Mode Distribution**
